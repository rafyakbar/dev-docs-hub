The badge component is used to render a small box with some text inside:

Copy

```

    New
</x-filament::badge>

```

## [‚Äã](#setting-the-size-of-a-badge)Setting the size of a badge

By default, the size of a badge is ‚Äúmedium‚Äù. You can make it ‚Äúextra small‚Äù or ‚Äúsmall‚Äù by using the `size` attribute:

```
    New
</x-filament::badge>

```

## [‚Äã](#changing-the-color-of-the-badge)Changing the color of the badge

By default, the color of a badge is ‚Äúprimary‚Äù. You can change it to be `danger`, `gray`, `info`, `success` or `warning` by using the `color` attribute:

```

    New

<x-filament::badge color="gray">
    New

<x-filament::badge color="info">
    New
</x-filament::badge>

<x-filament::badge color="success">

<x-filament::badge color="warning">
    New
</x-filament::badge>

```

## [‚Äã](#adding-an-icon-to-a-badge)Adding an icon to a badge

You can add an [icon](../styling/icons) to a badge by using the `icon` attribute:

```

    New
</x-filament::badge>

```

You can also change the icon‚Äôs position to be after the text instead of before it, using the `icon-position` attribute:
