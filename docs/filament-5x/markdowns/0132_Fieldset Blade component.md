## [​](#introduction)Introduction

You can use a fieldset to group multiple form fields together, optionally with a label:

Copy

```
<x-filament::fieldset>
    <x-slot name="label">
        Address
    </x-slot>
  
    {{-- Form fields --}}
</x-filament::fieldset>

```
